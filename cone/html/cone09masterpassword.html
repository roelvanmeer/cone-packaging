Master Passwords

Passwords for remote mail accounts are not usually saved, and they must be entered every time when opening the account. The "Master Password" option on the main menu replaces all remote account passwords with a single master password. The master password must still be entered every time when starting Cone. However, all other remote mail account passwords are saved, and used when opening the remote account. The master password is used to encrypt the file where remote mail accounts are saved.

The master password may be changed at any time, by selecting the option from the main menu. Answering "Y" to "Remove master password?" deletes all saved passwords, and mail account passwords must, once again, be entered each time when opening remote mail accounts.

A master password cannot be recovered, once forgotten. The only available option is to remove the master password. All saved passwords will be lost.
